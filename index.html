What is this?

It's a single HTML file with no build tools or external dependencies. You can drop it onto free hosting (GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.) and update the text anytime using their web editor.

  • One file, easy to edit

Quick Start

  1. Download this file as index.html.
  2. Create a free site on GitHub Pages, Netlify Drop, Vercel, or Cloudflare Pages.
  3. Upload index.html. Done!

Tip: To edit later, open the file in the host's web editor, change the text (look for markers), and save.
